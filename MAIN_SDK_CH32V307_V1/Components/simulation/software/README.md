# 控制算法模拟器-by嵌入式新起点 EXE 使用说明

## 1. 启动方式

直接双击软件打包后的 `.exe` 文件即可启动。

首次打开时会显示欢迎页，你可以：

- 点击 `开始使用` 进入主界面
- 取消勾选 `启动时显示欢迎页`，下次不再自动弹出

## 2. 快速使用

建议按下面顺序操作：

1. 在左侧 `设备` 页选择串口并点击 `连接`
2. 在左侧 `控制` 页设置算法、参考值和扰动等级
3. 点击 `启动` 开始输出仿真波形
4. 在中间查看波形，在右侧查看 3D 模型与运行状态
5. 需要保存数据时，点击 `开始录制`

## 3. 主要区域

- `设备`
  - 选择串口、波特率，完成连接和断开
- `控制`
  - 设置算法、参考值、扰动等级，发送预设命令
- `通道`
  - 勾选需要显示的波形通道
- `中间波形区`
  - 查看实时波形，可暂停、清空、一键聚焦、点击标记
- `右侧 3D / 状态区`
  - 观察姿态、场景和运行状态
- `底部控制台`
  - 查看日志输出

## 4. 常用操作

- `启动 / 停止`
  - 控制本地仿真波形是否开始输出
- `一键聚焦`
  - 让波形快速回到当前有效数据
- `点击标记`
  - 在波形上点击后显示该点的数值
- `波形窗口`
  - 支持小窗和全屏查看
- `设置`
  - 支持导出设置、加载设置、重置设置

## 5. 常见问题

- 没有波形输出
  - 请先点击 `启动`
- 无法连接设备
  - 检查串口号、波特率和设备是否已被其他软件占用
- 欢迎页不再显示
  - 可在 `帮助 -> 欢迎页` 手动再次打开
- 3D 模型未更新
  - 检查程序是否已启动，以及当前模型和场景是否切换正确
